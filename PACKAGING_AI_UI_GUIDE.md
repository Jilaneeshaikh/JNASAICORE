# Packaging AI Assistant UI Guide

## Visual Identity
- **Canvas Color**: Slate and Onyx deep palette (`#070B13` / `#0C111E`).
- **Accent Elements**: High-contrast, sharp borders, cyber-glow effects, and clean typography.
- **Typography**: Inter (UI / labels) paired with JetBrains Mono (data, specs, formulas, and terminal streams).

## UI Panel Layout
1. **Interactive Chat Container**: Chat canvas supporting code formatting, bullet points, source citations, and markdown.
2. **Context Inspector**: Collapsible HUD displaying exact variables from the active layout currently loaded in the `Context Builder`.
3. **Session Ledger / History Panel**: Right-docked or sidebar panel managing multiple workspace discussions.
4. **Validation Cockpit**: Run validations against rules with visual error thresholds and compliance bars.
5. **Knowledge Base Links**: Quick reference articles linked directly to ASTM databases.
