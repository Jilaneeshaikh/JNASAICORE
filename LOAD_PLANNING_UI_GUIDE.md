# Logistics UI & Workstation Guide

## Visual Principles
The interface utilizes a sleek, dark industrial palette optimized for low-latency workspace navigation and high mechanical legibility:
- **Primary Background**: Slate Black (`#070B13`)
- **Card Background**: Onyx Blue (`#0C111E`)
- **Accents**: Cyan-400 (`#22D3EE`) for capacities and highlight states; Indigo-400 (`#818CF8`) for assets.
- **Typography**: Inter for interface elements; JetBrains Mono for dimensions, cargo specs, audit tables, and XML streams.

## Core Interactive Views
1. **Logistics Dashboard**: Highlights key stats (staged plans, assigned cargo volume, total weight overhead, active revisions).
2. **Faceted Query Panel**: Allows rapid real-time list filtering by Plan number, Customer, Staging status, or Container equipment constraints.
3. **Capacity Gauge HUD**: Visualizes structural stack limits and volume budgets.
4. **Approval node console**: Allows quick sign-off or plan rejection with audit trail logs.
