# JNAS Changelog

All notable changes to the JNAS Enterprise Load Planning & Logistics Intelligence Platform will be documented in this file.

## [v1.6.0] - 2026-06-30
### Added
- **Enterprise Digital Thread Platform**: Centralized connectivity engine weaving every JNAS business, physical, and engineering object into a unified logical mesh.
- **Centralized Relationship Registry**: Full metadata modeling support (links strengths, owners, directionality, and custom rationales).
- **Visual Thread Map**: SVG-driven graphical map tracking connected objects, highlighting lineage and dependency nodes.
- **Hierarchy Tree & Timeline views**: Detailed top-down dependency structures paired with a chronological setup lineage ledger.
- **Recursive Traceability Solver**: Full forward (source to target), backward (root cause), and Change Impact assessment solver calculations.
- **Grounded Thread Copilot**: Interactive AI query simulator answering operational dependency impact questions on live graph values.
- **Workspace Isolation & Audits**: Role-Based Workspace Isolation (RBWI) and verified operator IP audit trails logging relationship edits.

## [v1.5.0] - 2026-06-30
### Added
- **Enterprise Decision Intelligence Platform**: Centralized decision orchestration layer for JNAS, serving as the strategic executive steering console.
- **KPI Registry Engine**: Structured indicators repository supporting metric codes, value trends, operational target goals, and automated formula bindings.
- **Steering Dashboards Directory**: Custom dashboard creator with role-based accessibility permissions, favorite state management, and direct KPI attachments.
- **Analytical Metrics Ledger**: Core operational metric aggregates from all JNAS workspace sub-modules (CRM, Projects, Engineering, Packaging, LMS, KMS).
- **AI Unified Context Fabric Grounding**: Deep alignment of active business standing properties in the grounding engine under the XML context node `<DECISION_INTELLIGENCE_COGNITION>`.
- **Grounding Decision Copilot**: Interactive query-and-recommendation simulator answering complex operational questions using live registry data.
- **Audit Trails**: Non-destructive logging of indicator adjustments and configuration changes with operator IP verification.

## [v1.4.0] - 2026-06-30
### Added
- **Enterprise Returnable Packaging Lifecycle & Asset Tracking Platform**: Fully implemented reusable asset registries for steel racks, plastic totes, kit carts, pallets, and custom fixtures.
- **Dynamic Lifecycle Engine**: Supports designed, manufactured, approved, released, in service, in transit, at customer, returned, maintenance, repair, and retired statuses.
- **Multi-Modal Tracking Layout**: Modular tracking configuration fields for RFID, BLE Beacons, QR, Barcode, and GPS devices.
- **Safety Inspection Panel**: Supports custom condition scoring, damage evaluations, next-inspection planning, and compliance pass/fail recommendations.
- **Maintenance Workspace**: Preventative routine maintenance and ad-hoc heavy repair tracking with financial cost indicators and part replacement lists.
- **AI Context Integration**: Linked active returnable database metrics to the dynamic JNAS Unified XML Context Fabric under the `<RETURNABLE_ASSET_COGNITION>` prompt node.
- **Omni-Command Center Bindings**: Connected the global workspace loader to trigger modal overlays for registration, tracking, and dashboard analysis.
- **Workspace Security & Isolation Level**: Role-based access and ledger audit trails.
- **Detailed Standards Documentation**: Generated full technical guides covering platforms, lifecycles, and inspections.
