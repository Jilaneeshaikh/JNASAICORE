# Logistics Workspace User Guide

The **Logistics Workspace** provides a comprehensive environment for packaging and logistics engineers to stage packaging designs and verify volumetric capacities before actual physical transport occurs.

## Functional Sections

### 1. Overview Tab
- Displays critical metadata about the plan, planner role, assigned vehicle types, and warehouse distribution facility targets.
- Captures links to active CRM customers and engineering projects.

### 2. Capacity & Load Model Tab
- Renders progress gauges showing estimated packaging volume (CBM), cumulative load weights, and total unit counts compared to container thresholds.
- Displays placeholders for loading zones and center-of-gravity (CoG) parameters for downstream 3D balance solvers.

### 3. Engineering Assets Tab
- Displays drawing numbers (`.step`, `.dxf`, or CAD references) that dictate physical tolerances.
- Directly resolves the design's Bill of Materials (BOM) components from the Packaging Studio.

### 4. Documents & Knowledge Tab
- Contextually links plans to ASTM D6199 and ISTA 3A transport testing specifications.
- Retrieves active Memory Engine records concerning clearance tolerances.

### 5. AI Co-pilot Sandbox
- Offers live consultations using the structured XML Context Fabric.
- Answers structural questions referencing standards.
